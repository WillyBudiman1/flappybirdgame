Cara Memainkan Game Flappy Birds

MOHON DIBACA SAMPAI SELESAI TUTORIALNYA!!!

  1. Klik tombol "Download Game Sekarang".
  2. Setelah didownload, klik "Mainkan Game Sekarang" untuk membuka website TurboWarp di tab baru.
  3. Di halaman TurboWarp, pilih "File" > "Load From Your Computer".
  4. Masukkan file yang sudah kamu download: "flappybirds tugas PG.sb3".
  5. Klik bendera hijau untuk memulai game-nya!
